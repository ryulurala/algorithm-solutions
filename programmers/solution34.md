## 문제 링크

[평균 구하기](https://programmers.co.kr/learn/courses/30/lessons/12944)

### C++

```cpp
#include <string>
#include <vector>

using namespace std;

double solution(vector<int> arr) {
    double answer = 0;

    for(auto e: arr){
        answer += e;
    }
    answer /= arr.size();

    return answer;
}
```

### JavaScript

```js
function solution(arr) {
  var answer = 0;

  arr.forEach((value) => {
    answer += value;
  });
  answer /= arr.length;

  return answer;
}
```
